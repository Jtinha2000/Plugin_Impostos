# Unturned Impostos

> Esse repositório contém uma extensão configurável do jogo Unturned. Nela é incrementado funcionalidades relacionadas a taxação da moradia dos jogadores, sendo todos os valores arrecadados direcionados ao armazenamento da cidade.

## ⚠️ Aviso

O projeto foi desenvolvido unicamente com fins recreativos, não há previsão para atualizações, futuros reparos nem qualquer garantia de qualidade ou segurança! Utilize a aplicação por sua conta e risco.
